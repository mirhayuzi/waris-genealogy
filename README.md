# Waris Genealogy 🧬

Waris Genealogy is a modern Android application designed to help users build, visualize, and manage their family tree easily.

## 🚀 Features (Planned)
- Create and manage family members
- Visual family tree structure
- Add photos and personal details
- AI-generated family history stories
- Islamic inheritance (Faraid) calculator

## 🛠 Tech Stack
- Kotlin (Android)
- Firebase (optional backend)
- Room Database (local storage)

## 📌 Vision
To become the #1 genealogy and Islamic inheritance app in Malaysia and globally.
