# 🎨 Waris Genealogy UI/UX Design

## 🧭 Design Concept
- Clean
- Modern
- Islamic-friendly aesthetic
- Simple for all ages

---

## 📱 Main Screens

### 1. Home Screen
- App logo (Waris Genealogy)
- Button: "Start Family Tree"
- Button: "View My Tree"
- Quick stats (Total family members)

---

### 2. Family Tree Screen
- Visual tree (nodes connected)
- Each node:
  - Profile photo
  - Name
- Tap → View details

### 3. Add Member Screen
- Name
- Gender
- Relationship
- Date of birth
- Photo upload
- Save button

---

### 4. Profile Detail Screen
- Full info
- Edit button
- Delete button
